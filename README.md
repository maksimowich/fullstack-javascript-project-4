# Загрузщик страниц #
## Установка: ##
```
  git clone https://github.com/dzencot/project-lvl3-s71 page_loader
  cd page_loader
  make install
```
## Использование: ##
```
  make run 1=http://your_page
```
Другой способ:
```
  make build
  node dist/bin/page_loader.js --help
```
